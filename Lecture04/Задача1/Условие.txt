Идея: совместная разработка
Система собирает информацию с датчиков, в ней есть модуль логирования,
который заносит в журнал все обращения к датчикам. В системе есть модуль,
выполняющий обращения для получения данных с датчиков и модуль генерации 
html- представления. Запуск системы осуществляется из головного модуля.

Модуль 1: сбор информации с датчиков
Модуль 2: логирование
Модуль 3: UI
Модуль 4: HTML-генератор
Модуль 5: главный модуль

Модуль 1: data_provider
Модуль 2: logger
Модуль 3: user_interface
Модуль 4: html_creater
Модуль 5: main
